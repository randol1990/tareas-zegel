# Instalación de dependencias
Los pasos para la instalación de dependencias  y ejecutar el proyecto son:
  1. Inicializar node en el directorio del proyecto: `npm init`
  2. Crear el directorio **node_modules** e instalar las dependencias: `npm install`
  3. Instalar **express**: `npm install express`
  4. Instalar **nodemon** de forma local: `npm install --save-dev nodemon`
  5. Inicializar servidor con el comando: `npm run dev` o `npm run start`
